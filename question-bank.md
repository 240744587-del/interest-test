# L1 儿童版（7-12岁，共30题）

> 设计原则：用"你更喜欢哪个"的二选一和场景选择，语言口语化，贴近小学生日常

## 第一部分：能量源泉（6题）

**Q1**（二选一）
放学后你有一整个下午的自由时间，你更想：
- A. 一个人安安静静地做自己喜欢的事 → [能量-独处充电] +2
- B. 找小伙伴一起玩 → [能量-社交充电] +2

**Q2**（二选一）
周末做了一整天的事，哪种情况你虽然累但很开心？
- A. 在外面跑了一天，打球/爬山/探险 → [能量-身体活动] +2
- B. 在家研究了一天喜欢的东西（拼乐高/画画/看书/玩游戏） → [能量-沉浸思考] +2

**Q3**（场景选择）
学校要搞一个大活动，你最想负责哪个部分？
- A. 设计海报和装饰教室 → [能量-创造表达] +3
- B. 上台主持或表演节目 → [能量-舞台展示] +3
- C. 帮老师安排流程，确保不出乱子 → [能量-组织管理] +3
- D. 想出一个超酷的活动点子 → [能量-构思策划] +3

**Q4**（二选一）
你更喜欢哪种作业？
- A. 有明确答案的（数学题、填空题） → [能量-确定性] +2
- B. 没有标准答案的（写作文、做手工、做实验） → [能量-开放性] +2

**Q5**（二选一）
如果可以选，你更想：
- A. 学一个全新的、从没接触过的东西 → [能量-新鲜感] +2
- B. 把已经会的东西练到超级厉害 → [能量-精通感] +2

**Q6**（场景选择）
下课十分钟，你通常在干嘛？
- A. 和同学聊天、打闹 → [能量-社交] +3
- B. 看课外书或者发呆想事情 → [能量-独处] +3
- C. 到处转转，去操场活动一下 → [能量-运动] +3
- D. 继续研究刚才课上感兴趣的内容 → [能量-求知] +3

## 驱动力 → 职业环境偏好
