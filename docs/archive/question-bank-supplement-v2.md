# 一、荣格心理功能维度补充题

> 这些题补充到各层级的"维度5：认知与决策风格"中
> 目标：从原来模糊的"发散vs聚焦/直觉vs分析"升级为荣格标准的S-N/T-F/J-P三组光谱

## L1 儿童版新增（替换原维度5的Q27-Q30，共6题）

**Q-Jung-L1-1**（二选一）[S-N 感觉-直觉]
老师讲了一个新知识，你更在意：
- A. 记住具体的事实和步骤（比如公式怎么用、步骤是什么） → [S 感觉] +2
- B. 搞懂为什么是这样，这个知识还能用在哪 → [N 直觉] +2

**Q-Jung-L1-2**（场景选择）[S-N 感觉-直觉]
写作文的时候，你更容易写出什么样的内容？
- A. 观察到的真实细节——颜色、声音、味道、发生了什么 → [S 感觉] +3
- B. 自己脑子里冒出来的故事和想法——如果这样会怎样？ → [N 直觉] +3
- C. 两种都有，看心情 → [平衡] +1
- D. 其实我不太喜欢写作文 → 不计分

**Q-Jung-L1-3**（二选一）[T-F 思维-情感]
两个好朋友吵架了都来找你，你会：
- A. 分析谁的理由更有道理，帮他们讲道理 → [T 思维] +2
- B. 先安慰他们的情绪，让他们都觉得被理解 → [F 情感] +2

**Q-Jung-L1-4**（二选一）[T-F 思维-情感]
看动画片的时候，你更关注：
- A. 剧情的逻辑——"这不合理啊！""为什么他要这样做？" → [T 思维] +2
- B. 角色的感受——"他好可怜""我好喜欢这个角色" → [F 情感] +2

**Q-Jung-L1-5**（二选一）[J-P 判断-知觉]
出去旅游，你更喜欢：
- A. 提前把每天要去哪里、吃什么都安排好 → [J 判断] +2
- B. 到了再说，想去哪去哪，随便逛 → [P 知觉] +2

**Q-Jung-L1-6**（二选一）[J-P 判断-知觉]
做作业的时候，你通常：
- A. 先做完再玩，做完心里才踏实 → [J 判断] +2
- B. 先玩一会，截止前再赶——反正总能做完 → [P 知觉] +2

## L2 少年版新增（替换原维度5的Q35-Q40，共8题）

**Q-Jung-L2-1**（量表题 1-5分）[S-N]

→ [N 直觉] 直接计分

**Q-Jung-L2-3**（量表题）[T-F]
"做决定时，我更看重客观的道理和逻辑推演，即使结果让某些人不开心。"
→ [T 思维] 直接计分

**Q-Jung-L2-4**（量表题）[T-F]
"做决定时，我会优先考虑大家的感受和关系，即使这意味着不是最'理性'的选择。"
→ [F 情感] 直接计分

**Q-Jung-L2-5**（量表题）[J-P]
"我喜欢提前计划好一切，不确定的事会让我焦虑。"
→ [J 判断] 直接计分

**Q-Jung-L2-6**（量表题）[J-P]
"我更喜欢保持灵活，讨厌被固定的计划束缚，最后一刻才做决定也没关系。"
→ [P 知觉] 直接计分

**Q-Jung-L2-7**（场景选择）[T-F]
考试成绩出来了，你的好朋友考砸了心情很差。你会怎么说？
- A. "别难过了，我们来分析一下哪里丢分了，下次改就好" → [T 思维] +3
- B. "我知道你很失望，没关系的，一次考试不代表什么" → [F 情感] +3
- C. "走吧，出去吃个好的，别想了" → [F 情感-行动型] +3
- D. "你其实进步了啊，上次你XX更差呢" → [T+F 混合] +1

**Q-Jung-L2-8**（场景选择）[S-N]
老师布置了一个"介绍你最喜欢的东西"的演讲，你会怎么准备？
- A. 列出具体的事实和细节——名称、特点、使用体验 → [S 感觉] +3
- B. 讲它对你意味着什么、它代表的理念和感觉 → [N 直觉] +3
- C. 准备一个有起承转合的故事来展示它 → [N 直觉-叙事型] +3
- D. 做一个图文并茂的PPT，用数据和图片说话 → [S 感觉-视觉型] +3

## L3 青年版新增（共8题）

**Q-Jung-L3-1**（量表题）[S-N]

→ [N 直觉] 直接计分

→ [T 思维] 直接计分

**Q-Jung-L3-4**（量表题）[T-F]
"我天然会考虑一个决定对每个相关人的情感影响——即使那不是决定的关键因素。"
→ [F 情感] 直接计分

**Q-Jung-L3-5**（量表题）[J-P]
"我享受把事情规划得井井有条的过程——日程表、to-do list、清单打勾让我满足。"
→ [J 判断] 直接计分

**Q-Jung-L3-6**（量表题）[J-P]
"我抗拒过早做决定——截止日期反而能激发我的最佳表现。"
→ [P 知觉] 直接计分

**Q-Jung-L3-7**（场景选择）[S vs N 综合]
如果可以选择研究方向，你更想做：
- A. 一个有明确步骤和可测量结果的实验 → [S 感觉] +3
- B. 一个没有前人做过、方向都要自己定义的开创性课题 → [N 直觉] +3
- C. 改良一个已有的方案，让它更高效 → [S+J] +3
- D. 把两个看似无关的领域连接起来，看看会发生什么 → [N+P] +3

**Q-Jung-L3-8**（场景选择）[T vs F 综合]
班级要决定旅行目的地，两个方案票数一样。你倾向于怎么决定？
- A. 客观比较两个方案的性价比、交通便利性、天气 → [T 思维] +3
- B. 看哪个方案能让更多人开心（尤其是少数派的感受） → [F 情感] +3
- C. 随机抽签——最公平 → [T 思维-公平型] +3
- D. 两个都去——分两次或者合并方案 → [P 知觉-整合型] +3

- C. 在结构化的框架内有一定灵活空间 → [J-P 平衡] +3
- D. 完全混沌，什么都要自己定义——前沿创业/研究型 → [P 极端知觉] +3

# 三、RIASEC一致性/区分度的计算补充题

> v2.0新增：为了更精确计算霍兰德六角模型的高级指标，
> 在L3/L4中增加迫选题（forced choice），减少社会赞许性偏差

## L3/L4通用（4道迫选题）

**Q-RIASEC-FC-1**（迫选：必须选最喜欢和最不喜欢的各一个）
以下活动中，你最想做的和最不想做的分别是？
- A. 拆解一台机器，搞清楚它怎么运转 → [R]
- B. 设计一个调查问卷，研究某个社会现象 → [I]
- C. 导演一出舞台剧 → [A]
- D. 在社区里组织一场志愿活动 → [S]
- E. 写一份商业计划书并说服投资人 → [E]
- F. 整理公司的全年财务数据并生成报告 → [C]
→ 最喜欢 +4，最不喜欢 -2

**Q-RIASEC-FC-2**（迫选）
以下工作环境，你最向往和最不想去的分别是？
- A. 户外工地/工坊/实验室（做实物） → [R]
- B. 大学/研究所/智库（做研究） → [I]
- C. 工作室/剧院/设计公司（做创作） → [A]
- D. 学校/医院/社区中心（服务人） → [S]
- E. 写字楼/创业公司/政府部门（推动事） → [E]
- F. 银行/会计所/档案馆（管理数据和流程） → [C]
→ 最向往 +4，最不想去 -2

**Q-RIASEC-FC-3**（迫选）
你最佩服哪种人？最不理解哪种人？
- A. 能徒手造出东西的工匠/工程师 → [R]
- B. 能解释世界运行规律的科学家 → [I]
- C. 能创造出打动人心的作品的艺术家 → [A]
- D. 能让别人变得更好的教育者/治愈者 → [S]
- E. 能白手起家建立事业的企业家 → [E]
- F. 能把复杂系统管理得井井有条的人 → [C]
→ 最佩服 +4，最不理解 -2

**Q-RIASEC-FC-4**（迫选）
如果只能选一种超能力：
- A. 一碰就能修好任何机器 → [R]
- B. 看一眼就能理解任何系统的运行原理 → [I]
- C. 脑中的画面能直接变成现实 → [A]
- D. 和任何人说一句话就能让ta安心 → [S]
- E. 说一段话就能让全场人跟你走 → [E]
- F. 一瞬间就能把任何数据整理成完美报表 → [C]
→ 选中 +3
