# 向野暖阳田野视觉设计

## 目标

在不改变题库、评分、隐私边界和页面流程的前提下，让“向野”呈现更温暖、成熟、有成长感的视觉气质。

## 已确认方向

- 背景采用“暖阳田野”：米白天空、柔和暖光、低饱和草木绿和抽象山野曲线。
- 去掉“向野”前的麦穗 emoji，不用具象品牌图标。
- 品牌字标采用“现代人文宋”：中文宋体为主，增加字距，搭配小号英文 `XIANGYE`。
- 首页装饰最完整；同意页、答题页、结果页使用更淡的版本，保证阅读和操作优先。
- 首页采用极简品牌首屏：只保留品牌字标、Slogan“从兴趣出发，向辽阔处生长”和醒目的“开始测评”按钮。
- 首屏不显示“免费 · 匿名 · 无需注册”标签，相关隐私说明统一放在下一屏，避免信息重复。

## 首页信息层级

1. 首屏集中展示“向野”、`XIANGYE`、Slogan“从兴趣出发，向辽阔处生长”、产品定位说明和“开始测评”按钮。
2. 产品定位说明为：“面向不同成长阶段的兴趣探索工具，帮助你更好地认识自己，发现值得尝试的成长方向。”
3. 点击“开始测评”跳转到独立的 `/start` 页面选择教育阶段。
4. 首页后续采用品牌叙事型长页，每个主要视口只讲一个特点：
   - 从多个角度，看见成长线索
   - 不同阶段，不同探索方式
   - 结果是线索，不是标签
   - 隐私优先，由你决定
5. 首页末屏再次展示“开始测评”行动入口。
6. `/start` 页面集中展示 L1-L4 教育阶段卡片，选择后进入原有同意与答题流程。

## 实现边界

- 只使用 CSS 渐变和伪元素，不新增图片、字体包或第三方请求。
- 保留现有页面布局、内容结构、交互和动效。
- 首页不展示教育阶段选择卡片；阶段选择属于独立 `/start` 页面。
- 不改变教育阶段选择后的同意、答题和报告流程。
- 主操作色从亮蓝调整为深橄榄绿；题目维度的功能性色彩仍保留。
- 打印报告保持白底，装饰背景不进入 PDF。

## 验收标准

- 首页不再出现麦穗字标。
- 首页首屏不出现“免费 · 匿名 · 无需注册”标签或四张说明卡片。
- 首页首屏具有明确的产品定位说明和唯一主要操作“开始测评”。
- 首页所有“开始测评”入口均跳转 `/start`。
- 首页不包含“选择教育阶段”或 L1-L4 阶段卡片。
- 四个产品特点分别使用独立的大段落呈现，不挤在同一组小卡片中。
- `/start` 页面完整展示四个阶段及题量、时长信息。
- 首页、同意页、答题页和报告页具有统一的暖色背景与草木绿主色。
- 手机端装饰不遮挡文字、按钮或选项。
- 现有测试、隐私检查、类型检查、代码检查和生产构建全部通过。
