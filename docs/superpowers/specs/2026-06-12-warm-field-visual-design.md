# 向野暖阳田野视觉设计

## 目标

在不改变题库、评分、隐私边界和页面流程的前提下，让“向野”呈现更温暖、成熟、有成长感的视觉气质。

## 已确认方向

- 背景采用“暖阳田野”：米白天空、柔和暖光、低饱和草木绿和抽象山野曲线。
- 去掉“向野”前的麦穗 emoji，不用具象品牌图标。
- 品牌字标采用“现代人文宋”：中文宋体为主，增加字距，搭配小号英文 `XIANGYE`。
- 首页装饰最完整；同意页、答题页、结果页使用更淡的版本，保证阅读和操作优先。
- 首页采用极简品牌首屏：只保留品牌字标、核心说明和醒目的“开始测评”按钮。
- 首屏不显示“免费 · 匿名 · 无需注册”标签，相关隐私说明统一放在下一屏，避免信息重复。

## 首页信息层级

1. 首屏占据主要视口高度，集中展示“向野”、`XIANGYE`、成长方向探索说明和“开始测评”按钮。
2. 点击“开始测评”后，页面平滑滚动到下一屏，不直接跳转测评。
3. 下一屏先展示六大理论、六个测评维度、隐私优先和 AI 解读可选等说明。
4. 教育阶段选择紧接说明区域展示，用户选中阶段后进入原有同意与答题流程。
5. 现有详细使用说明和理论名称保留在教育阶段选择之后。

## 实现边界

- 只使用 CSS 渐变和伪元素，不新增图片、字体包或第三方请求。
- 保留现有页面布局、内容结构、交互和动效。
- 首页允许调整内容顺序和纵向分屏，但不改变教育阶段选择后的流程。
- 主操作色从亮蓝调整为深橄榄绿；题目维度的功能性色彩仍保留。
- 打印报告保持白底，装饰背景不进入 PDF。

## 验收标准

- 首页不再出现麦穗字标。
- 首页首屏不出现“免费 · 匿名 · 无需注册”标签或四张说明卡片。
- 首页首屏具有唯一的主要操作“开始测评”，点击后准确滚动到下一屏说明区域。
- 理论、维度、隐私和 AI 说明在教育阶段选择之前完整可见。
- 首页、同意页、答题页和报告页具有统一的暖色背景与草木绿主色。
- 手机端装饰不遮挡文字、按钮或选项。
- 现有测试、隐私检查、类型检查、代码检查和生产构建全部通过。
