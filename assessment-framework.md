# 兴趣探索与职业启蒙测评系统 — 测评框架 v2.0

> v2.0 更新：系统性整合荣格心理类型、霍兰德六角模型、舒伯生涯发展理论、
> 施恩职业锚、以及国际/国内主流量表的设计精华。

---

## 零、理论地基

本测评系统不是从零发明，而是站在6大成熟理论之上的整合应用。
每个维度都有明确的理论出处和经过验证的测量范式。

### 0.1 荣格心理类型理论（Carl Jung, 1921）→ 维度1 + 维度5

荣格提出人的心理功能由两组维度构成：
- **心理能量方向**：外倾（Extraversion）vs 内倾（Introversion）——能量从外部世界获得还是内部世界获得
- **感知功能**：感觉（Sensing）vs 直觉（iNtuition）——关注具体事实还是可能性与模式
- **判断功能**：思维（Thinking）vs 情感（Feeling）——用逻辑还是价值观做决策
- **生活态度**：判断（Judging）vs 知觉（Perceiving）——偏好结构化还是开放灵活

MBTI 是荣格理论的通俗化应用（4维度 → 16型人格）。
本系统不做16型分类（过于标签化），而是保留荣格的**功能维度**作为连续光谱来测量，
融入"能量源泉"和"思维与行动风格"两个维度中。

**具体映射**：
| 荣格维度 | 本系统对应 | 测量方式 |
|---------|-----------|---------|
| E-I（外倾/内倾） | 维度1：能量源泉——社交充电 vs 独处充电 | 量表题 + 场景题 |
| S-N（感觉/直觉） | 维度5：思维风格——具象实感 vs 抽象直觉 | 量表题 |
| T-F（思维/情感） | 维度5：决策风格——逻辑分析 vs 价值共情 | 量表题 + 场景题 |
| J-P（判断/知觉） | 维度5：行动风格——结构计划 vs 灵活探索 | 量表题 |

**为什么用荣格而不是直接用MBTI**：
1. MBTI把连续光谱切成二元标签（你要么是E要么是I），丢失了大量信息
2. 本系统保留百分位光谱（你在E-I维度上的位置是35%，偏内倾但不极端）
3. 更适合发展性评估——孩子的类型偏好还在形成中，不应过早贴标签

### 0.2 霍兰德职业兴趣理论（John Holland, 1959）→ 维度3

霍兰德RIASEC六角模型是全球应用最广的职业兴趣框架。核心概念：
- 6种职业兴趣类型：R实际、I探究、A艺术、S社会、E企业、C常规
- **六角形排列**：相邻类型相关性高（如R-I、A-S），对角类型相关性低（如R-S、I-E）
- **一致性原则**：个人的前两三种类型代码越一致（在六角形上越相邻），职业满意度越高
- **区分度原则**：各类型得分差异越大，职业方向越清晰

**本系统的改进**：
1. 不只输出三字母代码（如RIA），还输出六角形上的完整雷达图
2. 计算一致性和区分度指标，告诉用户"你的方向有多清晰"
3. 对低区分度的用户（各类型得分接近）给出"你可能还在探索期"的诊断，而不是强行贴标签

**参考量表**：
- Strong Interest Inventory (SII)——霍兰德的标准实现，含6大兴趣主题 + 30个基本兴趣量表
- Self-Directed Search (SDS)——霍兰德本人开发的自我施测版
- 中国职业兴趣量表（龙立荣等）——对RIASEC进行了中国文化适应性修订

### 0.3 舒伯生涯发展理论（Donald Super, 1957）→ 年龄分层 + 维度4

舒伯的**生涯彩虹模型（Life-Career Rainbow）**提出：
- 人的一生扮演多种角色（儿童、学生、休闲者、公民、工作者、持家者）
- 生涯发展分5个阶段，每个阶段有不同的核心任务：

| 阶段 | 年龄 | 核心任务 | 本系统对应层级 |
|------|------|---------|-------------|
| 成长期（Growth） | 0-14岁 | 通过幻想、兴趣、能力形成自我概念 | L1（7-12）+ L2前半（13-14） |
| 探索期（Exploration） | 15-24岁 | 通过尝试形成暂定的职业偏好 | L2后半 + L3 + L4前半 |
| 建立期（Establishment） | 25-44岁 | 在选定领域稳定发展 | L4 |
| 维持期（Maintenance） | 45-64岁 | 保持已有地位，面对新挑战 | L4（扩展） |
| 衰退期（Decline） | 65+岁 | 逐步退出工作角色 | 暂不覆盖 |

**对本系统的关键影响**：
1. **L1（成长期）不应测"你适合什么职业"**，而是测"你的兴趣种子和能力苗头在哪"
2. **L2-L3（探索期）重点是拓宽视野**，报告应该鼓励"尝试"而不是"确定"
3. **L4（建立/维持期）可以给更具体的方向建议**，但也要考虑转型的可能

舒伯还提出**职业成熟度（Career Maturity）**的概念：
- 一个人对职业决策的准备程度
- 包括：自我认知、职业信息掌握、决策能力、现实取向
- 本系统在L3/L4中加入职业成熟度评估，帮助用户了解"你是否准备好做选择了"

**参考量表**：
- Career Maturity Inventory (CMI, Crites) ——职业成熟度经典量表
- Career Development Inventory (CDI, Super) ——舒伯本人开发
- 中学生职业成熟度量表（张智勇等）——国内适应版

### 0.4 施恩职业锚理论（Edgar Schein, 1978）→ 维度4

施恩发现人在工作10年后会形成一个稳定的"职业锚"——一种即使被迫也不愿放弃的核心价值。
8种职业锚：

| 职业锚 | 核心诉求 | 本系统"核心驱动力"映射 |
|-------|---------|-------------------|
| 技术/功能型（TF） | 在专业领域做到极致 | 探索驱动 + 精通驱动 |
| 管理型（GM） | 整合资源、领导他人 | 影响驱动 |
| 自主/独立型（AU） | 不受约束，自己定规则 | 自由驱动 |
| 安全/稳定型（SE） | 可预测、有保障 | 安全驱动 |
| 创业型（EC） | 创造新事业、新产品 | 创造驱动 + 影响驱动 |
| 服务型（SV） | 帮助他人、改善社会 | 连接驱动 |
| 挑战型（CH） | 攻克难题、超越极限 | 探索驱动 + 竞争驱动 |
| 生活方式型（LS） | 工作与生活的整合平衡 | 自由驱动 + 安全驱动 |

**对本系统的关键影响**：
- 对于L1-L2（中小学生），职业锚尚未形成，但可以检测"锚的萌芽"
- 对于L3-L4，可以更直接地测量职业锚倾向
- 本系统的"核心驱动力"维度是对施恩8种职业锚的简化整合

**参考量表**：
- Career Orientations Inventory (COI, Schein) ——施恩的标准职业锚量表
- 职业锚问卷（中文版，翁清雄等修订）

### 0.5 加德纳多元智能理论（Howard Gardner, 1983）→ 维度2

8种智能不再赘述（v1.0已覆盖），补充关键应用原则：
- 每个人都有8种智能，差异在于**相对强度组合**
- 智能可以发展，不是固定天赋——报告措辞应强调"当前倾向"而非"天生就是"
- 对中小学生（L1-L2）特别重要：传统教育只重视语言和逻辑数学两种智能，本系统要帮孩子发现其他6种被忽视的智能

**参考量表**：
- Multiple Intelligences Developmental Assessment Scales (MIDAS, Shearer)
- 中小学生多元智能评估量表（国内版，皮连生等）

### 0.6 心流理论 + 自我决定理论 + 盖洛普优势 → 维度1 + 维度4

**心流理论（Csikszentmihalyi, 1990）**：
- 当挑战与技能匹配时，人进入"心流"状态——完全沉浸、忘记时间、内在满足
- 心流频繁出现的活动领域往往指向天赋方向
- 本系统在维度1中测量心流体验的频率和触发场景

**自我决定理论（Deci & Ryan, 2000）**：
- 内在动机由3种基本心理需求驱动：自主感、胜任感、关联感
- 当一个方向同时满足这三者时，持续性最强
- 本系统在维度4中区分内在驱动 vs 外在驱动

**盖洛普优势理论（CliftonStrengths）**：
- 天赋 = 自然而然反复出现的思维、感觉和行为模式
- 天赋 ≠ 技能（技能可以后天训练，天赋是底层倾向）
- 本系统在维度2中寻找"无意识胜任区"——你觉得简单但别人觉得难的事

## 一、测评维度设计（6大维度，v2.0升级）

> v2.0 新增维度6：职业发展准备度（基于舒伯理论）

### 维度1：心理能量模式（Psychic Energy Pattern）
**测什么**：你的能量从哪来、怎么流动
**理论来源**：荣格E-I维度 + 心流理论 + 盖洛普优势
**子维度**：
- 1a. 能量方向：外倾充电 ←→ 内倾充电（荣格E-I）
- 1b. 心流触发：什么类型的活动让你进入忘我状态（心流理论）
- 1c. 能量消耗信号：什么让你耗电（反向指标，帮助排除不适合的方向）
- 1d. 刺激偏好：确定性偏好 ←→ 模糊性耐受（荣格J-P的能量面）

### 维度2：天然能力图谱（Natural Intelligence Profile）
**测什么**：你的8种智能的相对强度分布
**理论来源**：加德纳多元智能 + 盖洛普"无意识胜任"概念
**8个子维度**：语言、逻辑数学、空间、身体运动、音乐、人际、内省、自然观察
**测量策略**：
- L1-L2：用行为观察题（"你通常会做什么"）
- L3-L4：用自我评估量表 + 社会比较题（"别人觉得你哪方面强"）

### 维度3：兴趣类型图谱（RIASEC Interest Profile）
**测什么**：你的职业兴趣在六角模型上的分布
**理论来源**：霍兰德RIASEC + Strong Interest Inventory 的兴趣主题设计
**6个子维度**：R实际、I探究、A艺术、S社会、E企业、C常规
**进阶指标**（v2.0新增）：
- **一致性（Consistency）**：前两种类型是否在六角形上相邻（越相邻越一致）
- **区分度（Differentiation）**：最高分与最低分的差距（越大方向越清晰）
- **明确度（Clarity）**：整体得分分布的集中程度

| 一致性 | 区分度 | 解读 |
|--------|--------|------|
| 高 | 高 | 方向清晰，可以给具体建议 |
| 高 | 低 | 兴趣广泛，建议选择能整合多种兴趣的方向 |
| 低 | 高 | 内在张力——有两个不太相关的强兴趣，需要找到整合点 |
| 低 | 低 | 还在探索期，建议多尝试，不急着确定方向 |

### 维度4：核心驱动力（Core Drive / Career Anchor Seed）
**测什么**：内心深处最不愿放弃的核心价值
**理论来源**：施恩职业锚 + 自我决定理论 + 施瓦茨价值观
**7种驱动力**（v2.0 从6种扩展为7种）：
| 驱动力 | 对应施恩职业锚 | 核心诉求 |
|--------|-------------|---------|
| 创造驱动 | 创业型(EC) | 从无到有，留下作品 |
| 探索驱动 | 技术功能型(TF)+挑战型(CH) | 搞清楚真相，攻克难题 |
| 连接驱动 | 服务型(SV) | 与人深度连接，帮助他人 |
| 影响驱动 | 管理型(GM) | 领导他人，改变现状 |
| 自由驱动 | 自主独立型(AU) | 按自己的方式生活工作 |
| 安全驱动 | 安全稳定型(SE) | 可预期、有保障 |
| 平衡驱动 | 生活方式型(LS) | 工作与生活的整合 |

**年龄差异化处理**：
- L1-L2：测"驱动力萌芽"——通过偏好和幻想间接推断
- L3：测"驱动力倾向"——开始有自我觉察
- L4：测"职业锚雏形"——可以直接对标施恩理论

### 维度5：认知与决策风格（Cognitive & Decision Style）
**测什么**：你天然的信息处理和决策模式
**理论来源**：荣格四维度功能 + 认知风格研究
**4组连续光谱**：
| 光谱 | 荣格对应 | 左端 | 右端 |
|------|---------|------|------|
| 感知方式 | S-N | 具象实感（关注事实、细节、当下） | 抽象直觉（关注模式、意义、可能性） |
| 决策方式 | T-F | 逻辑分析（客观、原则、因果） | 价值共情（主观、和谐、人的感受） |
| 行动方式 | J-P | 结构计划（提前安排、按步骤来） | 灵活探索（随机应变、保持开放） |
| 协作方式 | — | 独立深耕（一个人效率高） | 团队协同（和人一起更好） |

**输出方式**：不给类型标签，给4条光谱上的位置（0-100），配合文字解读

### 维度6：职业发展准备度（Career Readiness）⭐ v2.0新增
**测什么**：你对做出职业/方向选择的准备程度
**理论来源**：舒伯Career Maturity + Crites CMI
**4个子维度**：
- 6a. 自我认知度：你对自己的兴趣、能力、价值观了解多少
- 6b. 职业信息度：你对各种职业方向了解多少
- 6c. 决策能力：你是否有成熟的决策框架
- 6d. 现实取向：你的期待是否与现实可操作

**为什么加这个维度**：
- 很多人测完兴趣后还是不会选，因为"知道自己喜欢什么"和"能做出好的选择"是两件事
- 这个维度帮助判断：用户需要的是"更了解自己"还是"学会怎么选"
- 对低成熟度的用户，报告侧重"继续探索"；对高成熟度的用户，报告给出具体行动方案

**年龄差异化**：
- L1：不测此维度（太小了）
- L2：只测自我认知度（"你知道自己喜欢什么吗"）
- L3：测自我认知 + 职业信息 + 现实取向
- L4：完整测4个子维度

## 二、年龄分层策略（对齐舒伯生涯发展阶段）

| 层级 | 年龄 | 舒伯阶段 | 核心任务 | 测评重心 | 题量 |
|------|------|---------|---------|---------|------|
| L1 | 7-12岁 | 成长期（幻想→兴趣子阶段） | 发现"我喜欢什么" | 兴趣种子+能力苗头+能量模式 | 30题 |
| L2 | 13-15岁 | 成长期→探索期过渡（兴趣→能力子阶段） | 开始知道"我擅长什么" | 全5维度+初步自我认知评估 | 40题 |
| L3 | 16-18岁 | 探索期（暂定子阶段） | 形成初步的职业偏好 | 全6维度+开放题+高考决策支持 | 48题 |
| L4 | 18+岁 | 探索→建立期 | 确认方向或发现需要转向 | 全6维度+工作能量审计+职业锚+开放题 | 55题 |

**关键原则（来自舒伯理论）**：
- ❌ 不对成长期（L1）的孩子说"你适合当XX"
- ✅ 对成长期说"你在XX方面有兴趣种子，可以多尝试"
- ❌ 不对探索期（L2-L3）强制确定方向
- ✅ 对探索期说"你目前的兴趣图谱是XX，建议体验这些方向"
- ✅ 只对建立期（L4成人）给出更具体的职业方向建议

## 三、计分机制

### 3.1 题目类型与分值
- **二选一题**（A/B选择）：选中对应维度 +2分
- **场景选择题**（4选1）：每个选项对应不同维度，选中 +3分
- **Likert量表题**（1-5分）：直接加到对应维度
- **排序题**（排出优先级）：第1名 +5分，第2名 +3分，第3名 +1分
- **开放题**：不计入标准分，作为AI生成报告的个性化素材

### 3.2 结果计算
1. 每个维度/子维度独立计算原始分
2. 原始分转换为百分位（0-100，参考常模）
3. 维度3（RIASEC）额外计算一致性、区分度、明确度
4. 维度6（准备度）独立计算，决定报告的"建议深度"
5. 综合画像 = 维度1×2×3×4×5 的交叉分析

### 3.3 信效度保障（v2.0新增）
- **内部一致性**：同一维度内多题交叉验证（如"社交充电"和"独处充电"应负相关）
- **社会赞许性控制**：部分题采用迫选法（forced choice），减少"我什么都好"的倾向
- **测谎/一致性检测**：含2-3对语义相反的题目，检测作答一致性
- **L1专属**：增加注意力检测题（如"请选择第三个选项"）防止低龄用户乱选

## 四、输出报告结构

### 4.1 你的心理画像
- 一句话画像（如"你是一个内倾-直觉型的、由探索驱动的创造者，在语言和逻辑智能上有天然优势"）
- 6大维度雷达图
- 荣格风格光谱图（4条0-100的滑块）
- RIASEC六角形雷达图

### 4.2 你的天赋地图
- Top 3 天然能力 + 具体表现场景
- 你的能量充电站 vs 耗电站
- 你的核心驱动力解读 + 与施恩职业锚的对应
- 你的认知决策风格解读

### 4.3 方向探索建议（按年龄和准备度定制）

**L1（成长期）报告侧重**：
- 你的兴趣种子在哪里（不说"你适合什么"）
- 推荐体验的活动/兴趣班/课外书
- 给家长的培育建议
- 关键句："这些种子需要浇水，而不是现在就被修剪成特定形状"

**L2（成长→探索过渡）报告侧重**：
- 你正在长出什么样的能力
- 推荐参加的社团/竞赛/体验营
- 初步的学科方向建议（文/理/艺/体/综合）
- 给家长的话：孩子的兴趣正在从幻想走向现实，请给ta试错的空间

**L3（探索期）报告侧重**：
- 你的RIASEC画像 + 一致性/区分度解读
- 匹配的大学专业方向（10-15个）
- 高考选科建议（新高考3+1+2/3+3参考）
- 职业发展准备度诊断：你是否准备好做选择了？
- "你可以做的3件事来验证你的方向"

**L4（建立期）报告侧重**：
- 完整的RIASEC + 职业锚 + 能量审计分析
- 当前方向匹配度评估（你现在做的是否符合你的底层驱动？）
- 匹配的职业领域 + 具体岗位（15-20个）
- 如果想转型的行动路线建议
- 被压抑的天赋方向诊断（基于开放题中的"嫉妒信号"和"理想一天"）

### 4.4 给家长/监护人的信（仅L1-L3）
- 孩子的天然能量模式和能力优势
- ✅ 建议支持和培育的方向
- ⚠️ 不建议强迫的方向（及原因）
- 家长可以做的具体行动（3-5条）
- 关键理念传达：舒伯理论的发展阶段观——"孩子现在的任务是探索，不是确定"

## 六、理论整合逻辑图

```
用户做题
  │
  ├─ 维度1：心理能量模式 ──────── 荣格(E-I) + 心流理论
  │    → 你的能量从哪来？什么让你进入心流？
  │
  ├─ 维度2：天然能力图谱 ──────── 加德纳多元智能 + 盖洛普优势
  │    → 你的8种智能强度分布？无意识胜任区在哪？
  │
  ├─ 维度3：兴趣类型图谱 ──────── 霍兰德RIASEC + SII/SDS
  │    → 你的兴趣六角形？一致性和区分度如何？
  │
  ├─ 维度4：核心驱动力 ─────────── 施恩职业锚 + 自我决定理论
  │    → 你最不愿放弃的核心价值是什么？
  │
  ├─ 维度5：认知决策风格 ──────── 荣格(S-N, T-F, J-P)
  │    → 你天然的信息处理和决策模式？
  │
  └─ 维度6：职业发展准备度 ───── 舒伯生涯发展理论 + CMI
       → 你是否准备好做方向选择了？
  │
  ▼
交叉分析 + AI个性化报告
  │
  ├─ 舒伯阶段 → 决定报告的语气和建议深度
  ├─ 维度6得分 → 决定建议的具体程度
  ├─ RIASEC一致性/区分度 → 决定方向推荐的确定性
  └─ 开放题内容 → AI生成个性化洞察
  │
  ▼
输出：分年龄、分准备度的个性化报告
```
